預備

具備以下知識有助於更滑順的理解此份報告內容

  • 任何一款靜態語言
  • Java 8+
  • TypeScript

條件

  • Kotlin 型別判定比 Java 嚴謹 : 能在編譯期就抓出淺在問題 -> NullPointerException
  • 支援 High Order Function : 可以撰寫更靈活的程式碼，在 Java 上要實現需要做很多工
  • 兼容 Java : 編譯後產生 class 檔一樣跑在 JVM 上，程式上也能和 Java 互相呼叫，對目前 incrte 技術親合度較高
